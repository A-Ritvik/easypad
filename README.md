# easypad
A repository for my custom macropad (easypad) containing a full 3d cad along with pcb, schematic, and code (used qmk firmware) files.
Features a rotary encoder, a 4*3 keyboard matrix , and a 0.91" OLED Display. Functionally, the rotary encoder can change volume; the keyboard matrix includes w/a/s/d/1/2/3/esc/enter/sleep/power on/shut down; and the display is currently set to display Capslock, numlock, scrollock, and the layout of the macropad.
