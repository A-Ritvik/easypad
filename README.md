# easypad
A repository for my custom macropad (easypad) containing a full 3d cad along with pcb, schematic, and code (used qmk firmware) files.
Features a rotary encoder, a 4*3 keyboard matrix , and a 0.91" OLED Display. Functionally, the rotary encoder can change volume; the keyboard matrix includes w/a/s/d/1/2/3/esc/enter/sleep/power on/shut down; and the display is currently set to display Capslock, numlock, scrollock, and the layout of the macropad.
# Full MacroPad 3d Model: 
Uses 4 M3 screws and heatset inserts to fit together. The case features a 15 degree tilt downwards, and is made of only 2 parts (the top and bottom all 3d printable)

# Bill of Materials:
(a full BOM.csv file can also be found under the production folder)
Here's everything you would need to build this: 
- 12x DSA Keycaps
- 12x Cherry_MX Switches
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
- 1x EC11 Rotary Encoder
- 1x Seeed XIAO RP2040 Microporcessor
- 12x 1N4148 DO-35 Diodes
- 1x 0.91" 128x32 OLED Display
- 1x Case (2 3D Printed Parts)
